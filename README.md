# FishingGame 2D
A project out of my own interest, to develop at home in my free time.

Inspired by Webfishing on Steam, I wanted to create a small 2D fishing game where the fish showed their sizes on screen.
This mechanic is interesting to me and feels funny while playing. For me, this game mechanic, especially when paired with a fishdex, gives the player a reason to keep fishing.
